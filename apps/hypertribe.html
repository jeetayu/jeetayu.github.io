HyperTRIBE Pipeline
Biswas Lab · MSKCC · End-to-end RBP target identification from FASTQ
Server-side pipeline. This app submits jobs to the local Mac server (192.168.68.37). FASTQ files must be accessible on that machine. The pipeline runs fastp → STAR → samtools → editing site calling → gene annotation → report.

Reference Files
Server-side paths to genome FASTA, STAR index, and GTF annotation. These do not need to be uploaded — they must already exist on 192.168.68.37.
STAR & genome references: Paths are pre-filled to the standard locations on the server. Genome FASTA and GTF must be downloaded from GENCODE, then the STAR index built with STAR --runMode genomeGenerate (~3 hours). FASTQ alignment will fail until these paths exist.

Sample Definition
Add ADARcd control samples and HyperTRIBE treatment samples. Each sample needs a unique name and paired-end FASTQ paths on the server.
FASTQ files must be gzip-compressed (.fastq.gz). Paired-end is required. Paths are on the server (192.168.68.37).
