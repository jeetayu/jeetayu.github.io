Deployment Notes

How Apps Are Served

All lab apps are pure JavaScript/HTML — no R, Shiny, or server-side processing required. To add a new app:

  1. Place the .html file in /Users/clawdbot/Sites/lab-website/apps/
  2. Add a card to this page linking to /apps/<appname>.html
  3. nginx serves it automatically as a static file
Server IP: 192.168.68.37 • Static files served by nginx • No R/Shiny process needed

Large data files (e.g. pre-cached gene expression JSON) live in /apps/gene-cache/ and are served as static assets.
