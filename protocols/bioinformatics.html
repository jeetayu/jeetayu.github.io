Calculating Microscopy Pixel Size

Determines the actual image pixel size in microns for quantitative microscopy analysis.

Required Parameters

  • Objective magnification (e.g., 60x)
  • Lens magnification — optional relay lens (1.25x, 1.6x, or 2x)
  • C mount value — typically 1x
  • Camera pixel size — actual sensor pixel dimensions in µm
  • Binning factor — 1×1, 2×2, or 4×4 (reduces noise at cost of resolution)

Formula

Image pixel size = camera pixel size × binning ÷ (objective mag × lens mag × C mount)

Example

Cascade 512 camera with 16 µm/pixel CCD, 60x objective, 1×1 binning, no relay lens:

16 µm × 1 ÷ (60 × 1 × 1) = 0.267 µm/pixel (266.7 nm/pixel)

Normalized BigWig from BAM

sorted_bam=$1; genome_fai_file=$2; prefix=${1%.bam*}
count=$(samtools view -c $sorted_bam)
norm_value=$(awk -v n=$count -v d=10000000 'BEGIN { printf "%.3f", (n/d) }')
bamToBed -split -i $sorted_bam > $prefix.bed
sort -k1,1 -k2,2n $prefix.bed > $prefix.sort.bed
genomeCoverageBed -bg -i $prefix.sort.bed -scale $norm_value -g $genome_fai_file > $prefix.bedgraph
bedGraphToBigWig $prefix.bedgraph $genome_fai_file $prefix.bw
rm $prefix.bed $prefix.sort.bed $prefix.bedgraph

Use the BigWig Normalization Calculator for the scale factor.

rMATS — Alternative Splicing

# Standard command (hg19, paired-end, novel splice sites)
rmats.py \
  --b1 control_bams.txt \
  --b2 experimental_bams.txt \
  --gtf /path/to/hg19/genes.gtf \
  --bi /path/to/hg19_star_index \
  --od output_dir/ \
  -t paired \
  --novelSS \
  --variable-read-length \
  --allow-clipping \
  --readLength 150 \
  --cstat 0.0001 \
  --libType fr-firststrand \
  --tmp output_tmp/

libType options: fr-firststrand (TruSeq stranded), fr-unstranded (unstranded or older data), single for single-end.

Ligandability vs Reactivity

Reactivity profiling: probe added to lysate — measures all cysteines accessible in vitro.
Ligandability profiling: probe added to live cells or intact cell lysate with compound pre-treatment — measures pharmacologically relevant cysteines.

GSEA — Gene Set Enrichment Analysis

Four input files required:

  1. Expression dataset (.gct or .txt): normalized RNA-seq counts
  2. Phenotype labels (.cls file): sample class assignments
  3. Gene sets (.gmt): download from MSigDB (hallmarks, KEGG, GO)
  4. Chip annotation (.chip): gene symbol ↔ probe ID mapping
Use normalized (not raw) read counts from STAR GeneCounts or DESeq2 vst-normalized values.
